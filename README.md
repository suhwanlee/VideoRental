# VideoRental


2023 메디슨 경력 과제

고객이 영화를 대여하고, 영수증을 발급하는 프로그램이며, 아래의 기능을 담고있습니다.
    - 입력받은 영화를 원하는 날짜만큼 대여가능합니다.
    - 고객이 대여할 영화는 영화제목과 장르를 입력받아 작성됩니다.
    - 대여 시 대여금, 포인트가 영화 종류 마다 다르게 책정됩니다.
    - 출력되는 영수증에는 영화별 가격, 합산가격, 획득한 포인트가 보여집니다.
